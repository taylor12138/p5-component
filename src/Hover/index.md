# Hover

## 怪盗字体动效
## 案例演示

```jsx
import { Font } from 'p5-react-component';
import '../global.less';

export default () => <div>
    <span class="p5-hover-animation-gray">触发hover特效 gray</span>
    <div />
    <span class="p5-hover-animation-mix">触发hover特效 mix</span>
    <Font>
        <span class="p5-hover-animation-mix">我是怪盗，这是我的预告函，谢谢</span>
    </Font>
</div>
```
